# ExamPrep Studio

Refined full-stack dashboard for student exam preparation.

## Why this version is better
- More natural project structure
- Feature-based frontend modules
- Smaller backend files with service helpers
- Cleaner copy and less template-like UI text
- Responsive layout for desktop, tablet and mobile

## Stack
- React + Vite + React Router + Recharts
- Express + SQLite

## Local run
### Backend
```bash
cd backend
npm install
npm run seed
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
